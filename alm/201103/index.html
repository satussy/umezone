仕組み

  • セレクタを駆使
    • ユーザ行動擬似セレクタの:focus
    • 直接隣接結合子 E + E
  • 要素がフォーカスを持っている時にその直後の要素のスタイルを定義することが可能

解説:ページ切り替え

HTML
<input class="dummy"></input>
<div class="page">１ページ目</div>

<input class="dummy"></input>
<div class="page">２ページ目</div>

<input class="dummy"></input>
<div class="page">３ページ目</div>
CSS
.page { opacity:0;}
↑基本的に非表示
.dummy:focus + .page { opacity:1 ; }
↑直前のinputにフォーカスがあるときだけ表示

解説:切り替え効果 1/2

  • transition(-webkit-transition)
    • スタイルのプロパティに変化があった場合現在の値から、新しい値へ時間をかけて変化させる

解説:切り替え効果 2/2

.page { 
    -webkit-transition :
        opacity
        0.3s 
        linear
    ;
    opacity:0;
}
.dummy:focus + .page { opacity:1 ; }
  • スライドが切り替わると opacityが 0 から 1 へ
    • この時切り替え効果
      • opacityを
      • 0.3秒かけて
      • 0から1へと変化

切り替え効果サンプル1

div.page.slide_transition_1 { 
  -webkit-transition : -webkit-transform 1s ease-out ;
  -webkit-transform : translateX( 1024px );
}
.dummy:focus + div.page.slide_transition_1 { -webkit-transform : translateX( 0 ); }

切り替え効果サンプル2

div.page.slide_transition_2 { 
    -webkit-transition : -webkit-transform 1s linear ;
    -webkit-transform : scaleY( 0.00001 );
}
.dummy:focus + div.page.slide_transition_2 { -webkit-transform : scaleY( 1 ); }

切り替え効果サンプル3

div.page.slide_transition_3 { 
    opacity:0;
}
.dummy:focus + div.page.slide_transition_3 { 
    -webkit-animation: slide_transition_3 1000ms ease-in 1;
    -webkit-transform : translateY( 0 );
    opacity:1;
}

@-webkit-keyframes slide_transition_3 {
    0% { -webkit-transform:translateY( -768px ); }
   80% { -webkit-transform:translateY(    0   ); }
   90% { -webkit-transform:translateY(  -40px ); }
  100% { -webkit-transform:translateY(    0   ); }
}
